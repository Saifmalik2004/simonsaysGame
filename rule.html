Rules of the Game

Sequence Display:

The game starts by displaying a sequence of one color.
Each level, one more color is added to the sequence.

User Input:

        After the sequence is displayed, you must repeat it by clicking the buttons in the correct order.
        If you repeat the sequence correctly, you move to the next level.
        If you make a mistake, the game ends.

Progression:

The sequence gets longer and harder as you progress through the levels.
        Your goal is to see how many levels you can complete.

Game Over:

repreWhen you make a mistake, the game will play a "Game Over" sound.
Your score (the level you reached) will be displayed.
You can restart the game by pressing any key.
